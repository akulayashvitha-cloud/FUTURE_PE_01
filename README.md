# FUTURE_PE_01# Task 1 - AI Website Copy Generator

## 📌 Business Chosen
GlowCare Skin Clinic (Hyderabad)

## 🎯 Objective
To generate high-quality website copy for a local business using AI and prompt engineering.

## 🛠️ Tools Used
- ChatGPT
- GitHub

## 🧠 Prompt Strategy
A structured prompt was designed to generate:
- Homepage content
- Services section
- Call-to-action (CTA)

The prompt ensures:
- Clear value proposition
- Business-specific content
- Conversion-focused writing

## 📄 Output
Generated complete website copy including:
- Homepage (headline, subheadline, intro)
- Services (with descriptions)
- CTA section

## ✨ Key Features
- Simple and professional language
- Strong and clear CTAs
- Non-generic content
- Ready to use for real websites

## 📁 Files Included
- prompt.txt → Contains AI prompt
- output.txt → Contains generated website content
