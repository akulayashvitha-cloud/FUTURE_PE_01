# 🚀 AI Website Copy Generator (Task 1)

## 📌 Project Overview
This project demonstrates how AI and Prompt Engineering can be used to create **high-converting website copy** for real local businesses.

The goal is to solve a common problem:
> Many businesses lose customers due to unclear, weak, or generic website content.

This project transforms that using **AI-generated, conversion-focused copy**.

---

## 🏥 Business Chosen
**GlowCare Skin Clinic (Hyderabad)**

A local skincare clinic offering treatments for:
- Acne  
- Hair fall  
- Skin brightening  
- Anti-aging  

---

## 🎯 Objective
To design a **structured prompt system** that generates:

- Clear and compelling homepage content  
- Service descriptions tailored to the business  
- High-converting Call-To-Action (CTA) sections  
- Website-ready content for real deployment  

---

## 🧠 Prompt Strategy

A reusable prompt framework was created with:

- Defined business inputs (name, location, services)  
- Conversion-focused structure  
- Tone control (professional + friendly)  
- Clear output formatting  

### ⚙️ Framework Logic:
This ensures:
✔ Personalized content  
✔ Strong value proposition  
✔ Real-world usability
